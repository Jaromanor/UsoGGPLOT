Echa un vistazo a la tabla anterior. Muestra las primeras seis líneas de nuestros datos de ejemplo, y que nuestros datos están constituidos por tres columnas.

El grupo y el subgrupo de columnas son indicadores de grupo, y el valor de la columna contiene valores numéricos distribuidos aleatoriamente.

También tenemos que instalar y cargar el paquete ggplot2 , por si queremos usar las funciones que vienen incluidas en el paquete:

Como siguiente paso, podemos graficar nuestros datos:

ggplot ( datos,                           # Dibujar ggplot2 boxplot con posición dodge 
       aes ( x =  grupo ,
           y =  valor ,
           relleno = subgrupo ) )  + 
  geom_boxplot ( posición = position_dodge ( ancho =  0.7 ) )

ggplot(data, # Draw ggplot2 boxplot with position dodge aes(x = group, y = value, fill = subgroup)) + geom_boxplot(position = position_dodge(width = 0.7))

 

 

En la Figura 1 se muestra que hemos logrado crear un diagrama de caja ggplot2 con especificaciones de sobreexposición de posición (es decir, las cajas se superponen ligeramente entre sí).

Supongamos que también queremos mostrar nuestros puntos de diagrama de caja con un cierto nivel de fluctuación. Entonces, podríamos intentar usar las funciones geom_point y position_jitter como se muestra a continuación:

ggplot ( datos,                           # Jitter no funciona correctamente 
       aes ( x =  grupo ,
           y =  valor ,
           relleno = subgrupo ) )  + 
  geom_boxplot ( )  + 
  geom_point ( position = position_jitter ( ancho =  0.1 ) )

ggplot(data, # Jitter does not work properly aes(x = group, y = value, fill = subgroup)) + geom_boxplot() + geom_point(position = position_jitter(width = 0.1))

 

 

Como se muestra en la Figura 2, hemos creado una nueva versión de nuestro diagrama de caja, donde hemos agregado puntos con fluctuaciones en la parte superior de las cajas.

Sin embargo, desafortunadamente, todos los puntos con fluctuaciones están centrados alrededor de la mitad de cada grupo; los diferentes subgrupos no se reflejan.

En el siguiente ejemplo, le mostraré cómo evitar este problema. ¡Así que sigue leyendo!

 

Ejemplo: Dibujar ggplot2 Boxplot con Jitter & Position Dodge usando la función position_jitterdodge()

En este ejemplo, demostraré cómo usar el jitter y el esquivar la posición simultáneamente.

Para esta tarea, podemos usar la función position_jitterdodge dentro de la función geom_point.

Dentro de la función position_jitterdodge, podemos especificar varios argumentos como jitter.width y dodge.width.

Asegúrese de que el argumento dodge.width dentro de position_jitterdodge tenga el mismo valor que el ancho de position_dodge dentro de la función geom_boxplot (es decir, 0.7).

El argumento jitter.width se puede especificar según sus preferencias personales.

Dibujemos nuestro gráfico:

ggplot ( datos,                           # ggplot2 boxplot con jitter y dodge 
       aes ( x =  grupo ,
           y =  valor ,
           relleno = subgrupo ) )  + 
  geom_boxplot ( posición = position_dodge ( ancho =  0.7 ) )  + 
  geom_point ( posición = position_jitterdodge ( jitter . ancho  =  0.1 ,
                                             esquivar _ ancho  =  0.7 ) )

ggplot(data, # ggplot2 boxplot with jitter & dodge aes(x = group, y = value, fill = subgroup)) + geom_boxplot(position = position_dodge(width = 0.7)) + geom_point(position = position_jitterdodge(jitter.width = 0.1, dodge.width = 0.7))

 

 

En la Figura 3, puede ver que hemos dibujado un diagrama de caja ggplot2 con fluctuación de fase y sobreexposición de posición juntos.

También podemos ajustar el diseño de los puntos para que nuestro gráfico sea aún más bonito. En la siguiente sintaxis R, estoy configurando el color de relleno de los puntos con fluctuaciones para que sea igual a los colores del subgrupo, y estoy cambiando el argumento pch para mostrar un borde negro alrededor de los puntos:

ggplot ( datos,                           # Cambiar los colores de los puntos de fluctuación 
       aes ( x =  grupo ,
           y =  valor ,
           relleno = subgrupo ) )  + 
  geom_boxplot ( posición = position_dodge ( ancho =  0.7 ) )  + 
  geom_point ( posición = position_jitterdodge ( jitter . ancho  =  0.1 ,
                                             esquivar _ ancho  =  0.7 ) ,
             aes ( relleno = subgrupo ) ,
             pc =  21 )

ggplot(datos, # Cambiar los colores de los puntos de fluctuación aes(x = grupo, y = valor, relleno = subgrupo)) + geom_boxplot(posición = position_dodge(ancho = 0.7)) + geom_point(posición = position_jitterdodge(jitter.width = 0.1, dodge.width = 0.7), aes(fill = subgrupo), pch = 21)

 

 

Como se muestra en la Figura 4, hemos creado una versión actualizada de nuestro diagrama de caja de esquivar la posición y la fluctuación de fase donde se ha cambiado el color y el estilo de los puntos.

¡Se ve muy bien!
