Marcas del eje ggplot2: una guía para personalizar marcas y etiquetas

El objetivo de este tutorial es describir cómo personalizar las marcas y etiquetas de los ejes en el software R usando el paquete ggplot2 .


Libro relacionado:


GGPlot2 Essentials para una gran visualización de datos en R

Datos

Los datos de ToothGrowth se utilizan en los ejemplos siguientes.

# Convert dose column from numeric to factor variable
ToothGrowth$dose <- as.factor(ToothGrowth$dose)
head(ToothGrowth)
##    len supp dose
## 1  4.2   VC  0.5
## 2 11.5   VC  0.5
## 3  7.3   VC  0.5
## 4  5.8   VC  0.5
## 5  6.4   VC  0.5
## 6 10.0   VC  0.5

Asegúrese de que la columna de dosis se convierta como un factor utilizando el script R anterior.

Ejemplo de parcelas

library(ggplot2)
p <- ggplot(ToothGrowth, aes(x=dose, y=len)) + geom_boxplot()
p

Cambiar la apariencia de las etiquetas de marca de graduación del eje

El color , el tamaño de fuente y la fuente de las etiquetas de marca de verificación del eje se pueden cambiar usando las funciones theme() y element_text() de la siguiente manera:

# x axis tick mark labels
p + theme(axis.text.x= element_text(family, face, colour, size))
# y axis tick mark labels
p + theme(axis.text.y = element_text(family, face, colour, size))

Los siguientes argumentos se pueden usar para que la función element_text() cambie la apariencia del texto:


  • familia : familia de fuentes
  • cara : cara de la fuente. Los valores posibles son "simple", "cursiva", "negrita" y "negrita.cursiva"
  • color : color del texto
  • tamaño : tamaño del texto en pts
  • ángulo : ángulo (en [0, 360])


# Change the appearance and the orientation angle
# of axis tick labels
p + theme(axis.text.x = element_text(face="bold", color="#993333", 
                           size=14, angle=45),
          axis.text.y = element_text(face="bold", color="#993333", 
                           size=14, angle=45))

Ocultar etiquetas de marcas de verificación de los ejes x e y

Las marcas de eje y las etiquetas de marca de marca se pueden eliminar usando la función element_blank() de la siguiente manera:

# Hide x an y axis tick mark labels
p + theme(
  axis.text.x = element_blank(),
  axis.text.y = element_blank())
# Remove axis ticks and tick mark labels
p + theme(
  axis.text.x = element_blank(),
  axis.text.y = element_blank(),
  axis.ticks = element_blank())

Cambiar líneas de eje

Las líneas del eje se pueden cambiar usando la función element_line() de la siguiente manera:

p + theme(axis.line = element_line(colour, size, linetype,
                                   lineend, color))

Los argumentos de element_line() son:


  • color , color : color de línea
  • tamaño : tamaño de línea
  • tipo de línea : tipo de línea. El tipo de línea se puede especificar usando texto ("en blanco", "sólido", "discontinuo", "punteado", "punteado", "guión largo", "dos guiones") o números (0, 1, 2, 3, 4, 5, 6). Tenga en cuenta que tipo de línea = "sólido" es idéntico a tipo de línea = 1. Los tipos de línea disponibles en R se describen en esta publicación: Tipo de línea en el software R
  • lineend : fin de línea. Los valores permitidos para el final de la línea son: "redondo", "a tope" o "cuadrado"


# Change the line type and color of axis lines
p + theme( axis.line = element_line(colour = "darkblue", 
                      size = 1, linetype = "solid"))

Establecer marcas de eje para ejes discretos y continuos

El eje x o y puede ser discreto o continuo. En cada uno de estos dos casos, las funciones que se utilizarán para establecer las marcas de los ejes son diferentes.

Personalizar un eje discreto

Las funciones scale_x_discrete() y scale_y_discrete() se utilizan para personalizar los ejes x e y discretos, respectivamente.

Es posible utilizar estas funciones para cambiar los siguientes parámetros del eje x o y:

  • títulos de eje
  • límites del eje (rango de datos para mostrar)
  • elegir dónde aparecen las marcas de verificación
  • etiquetar manualmente las marcas de verificación

Los formatos simplificados de scale_x_discrete() y scale_y_discrete() son:

scale_x_discrete(name, breaks, labels, limits)
scale_y_discrete(name, breaks, labels, limits)

  • nombre : etiquetas de eje x o y
  • breaks : controle los descansos en la guía (ticks de eje, líneas de cuadrícula, …). Entre los valores posibles, se encuentran:
    • NULL : ocultar todos los descansos
    • waiver() : el cálculo de descanso predeterminado
    • un carácter o vector numérico que especifica qué saltos mostrar
  • etiquetas : etiquetas de marcas de graduación del eje. Los valores permitidos son:
    • NULL sin etiquetas
    • renuncia () para las etiquetas predeterminadas
    • vector de caracteres que se utilizará para romper etiquetas
  • límites : un vector de caracteres que indica el rango de datos


Tenga en cuenta que, en los ejemplos a continuación, usaremos solo las funciones scale_x_discrete() y xlim() para personalizar las marcas de verificación del eje x. El mismo tipo de ejemplos se puede aplicar a un eje y discreto usando las funciones scale_y_discrete() y ylim() .

Cambiar el orden de los elementos

El argumento limites se utiliza para cambiar el orden de los elementos:

# default plot
p
# Change the order of items
# Change the x axis name
p + scale_x_discrete(name ="Dose (mg)", 
                    limits=c("2","1","0.5"))

Cambiar etiquetas de marca de verificación

El nombre de los textos de marca de verificación se puede cambiar de la siguiente manera:

# Solution 1
p + scale_x_discrete(breaks=c("0.5","1","2"),
        labels=c("Dose 0.5", "Dose 1", "Dose 2"))
# Solution 2 : same plot as solution 1
p + scale_x_discrete(labels=c("0.5" = "Dose 0.5", "1" = "Dose 1",
                              "2" = "Dose 2"))

Elija qué elementos mostrar

El siguiente código R muestra el diagrama de caja para el primer elemento (dosis = 0,5) y el último elemento (dosis = 2):

# Solution 1
p + scale_x_discrete(limits=c("0.5", "2"))
# Solution 2 : same result as solution 1
p + xlim("0.5", "2")

Personalizar un eje continuo

Las funciones scale_x_continuous() y scale_y_continuous() se utilizan para personalizar los ejes x e y continuos, respectivamente.

Usando estas dos funciones, se pueden modificar los siguientes parámetros del eje x o y:

  • títulos de eje
  • límites de los ejes (establecer el mínimo y el máximo)
  • elegir dónde aparecen las marcas de verificación
  • etiquetar manualmente las marcas de verificación

Los formatos simplificados de scale_x_continuous() y scale_y_continuous() son:

scale_x_continuous(name, breaks, labels, limits, trans)
scale_y_continuous(name, breaks, labels, limits, trans)

  • nombre : etiquetas de eje x o y
  • breaks : controle los descansos en la guía (ticks de eje, líneas de cuadrícula, …). Entre los valores posibles, se encuentran:
    • NULL : ocultar todos los descansos
    • waiver() : el cálculo de descanso predeterminado
    • un carácter o vector numérico que especifica las pausas para mostrar
  • etiquetas : etiquetas de marcas de graduación del eje. Los valores permitidos son:
    • NULL sin etiquetas
    • renuncia () para las etiquetas predeterminadas
    • vector de caracteres que se utilizará para romper etiquetas
  • límites : un vector numérico que especifica los límites del eje x o y (mín, máx)
  • trans para transformaciones de eje. Los valores posibles son "log2", "log10", "sqrt", etc.


Estas funciones se pueden utilizar de la siguiente manera:

# scatter plot
sp<-ggplot(cars, aes(x = speed, y = dist)) + geom_point()
sp
# Change x and y axis labels, and limits
sp + scale_x_continuous(name="Speed of cars", limits=c(0, 30)) +
  scale_y_continuous(name="Stopping distance", limits=c(0, 150))

Establecer la posición de las marcas de verificación

El siguiente código R establece la posición de las marcas en el eje y del diagrama de caja. La función scale_y_continuous() y los saltos de argumento se utilizan para elegir dónde aparecen las marcas:

# Set tick marks on y axis
# a tick mark is shown on every 5
p + scale_y_continuous(breaks=seq(0,40,5))
# Tick marks can be spaced randomly
p + scale_y_continuous(breaks=c(5,7.5, 20, 25))
                     
# Remove tick mark labels and gridlines
p + scale_y_continuous(breaks=NULL)

Dar formato al texto de las etiquetas de marca de verificación

Las etiquetas de marcas de verificación se pueden formatear para que se vean como porcentajes, dólares o notación científica. Se requiere la balanza del paquete .

library(scales)
# Format labels as percents
p + scale_y_continuous(labels = percent)
# Format labels as scientific
p + scale_y_continuous(labels = scientific)

Los valores posibles para las etiquetas son coma, porcentaje, dólar y científico. Para más ejemplos, lea la documentación del paquete scales : ?scales::trans_new
